弹性消除算法模型: 使用基础值吸引比它小的值，直到吸收全部
Destroying Asteroids question
BaseAmount = 10
Amounts = {1,5,9,12,15,20}

public void canDestroy() {
   Arrays.sort(amounts);
   long currentAmount = baseAmount;
   for (int amount : amounts) {
       if (currentAmount < amount) {
           return false;
       }
       currentAmount += amount;
   }
   return true;
}
