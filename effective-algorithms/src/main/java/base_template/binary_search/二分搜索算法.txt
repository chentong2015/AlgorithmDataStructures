TODO. 使用场景: 是否需要查找或定位
1. 从有序的数据中找到指定数据，不需遍历所有值，时间复杂度为O(log(n))
2. 从无序的数据中找到指定数据，必须在二分位计算后能够判断前后移动方向
3. 从拼接好的排序堆片段数据进行二分法查找 O(2log(n)) = O(log(n))

- 二分搜查时可以实时计算"二分点两侧"结果(总值范围 & 距离差值)
- 二分搜索偏向搜索: 找到median值后不一定返回，继续偏向最左值或最右值
- 二分搜索片段判断: 找到median值后根据区间片段判断移动([mid]-[mid+k])
