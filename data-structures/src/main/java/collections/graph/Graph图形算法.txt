TODO: 图问题的解决算法
1. Breadth First Search 广度优先遍历，遍历完图形中的节点
2. Depth First Search   深度优先遍历，找到图形中的目标节点
3. Node InDegree        根据到结点的指向数目，依次从低到高，遍历节点之后更新相关节点计数
4. Kahn's Algorithm     拓扑排序算法(Topological Sorting)
